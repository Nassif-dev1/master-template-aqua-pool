# AquaPro Pool Services — Master Template

A premium, professional pool maintenance website template built for local service businesses in Houston, TX.

## Pages

- **index.html** — Homepage (hero, trust bar, services, how-it-works, testimonials, quote form, service areas)
- **services.html** — Detailed services page (6 service blocks with CTAs)
- **about.html** — About Us (company story, team, certifications)
- **contact.html** — Contact page (form, map, emergency banner)

## Design

- Deep navy/aqua blue color palette
- Manrope typography
- Fully responsive desktop layout
- Trust badges (BBB, CPO, Insurance)
- Multiple CTAs for lead generation

## Built With

- HTML/CSS (generated via Google Stitch)
- Ready to deploy on Vercel, Netlify, or GitHub Pages
